Language Machines Events Documentation
======================================

This Project demostrates how event detection and time-to-event estimation can be applied on social media in the context of 
`ADNEXT <http://www.commit-nl.nl/projects/wp-packages/adaptive-information-extraction-over-time-adnext/>`_, 
`INFITI <http://www.commit-nl.nl/projects/information-retrieval-for-information-services/>`_, 
`COMMIT <http://www.commit-nl.nl//>`_,  project.
