Language Machines Events Documentation¶

This Project demostrates how event detection and time-to-event estimation can be applied on social media in the context of ADNEXT, INFITI, COMMIT, project.

The event detection is a result of the research done by Florian Kunneman.

The time-to-event estimation module developed by Ali Hürriyetoglu.
